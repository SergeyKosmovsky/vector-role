Role vector
=========

Репа для установки программы Vector в рамках ДЗ площадки netology

Requirements
------------


Centos8

Role Variables
--------------

Указание весрии в дефолтных настройках, текущая: "0.22.1-1"

Example Playbook
----------------

```
- hosts: servers
  roles:
    - vector-role
```
